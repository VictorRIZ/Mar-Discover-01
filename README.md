# Mar-Discover-01

Uma aplicação voltada para a área de finanças. Nessa aplicação, é possível 
inserir valores positivos e negativos, e no fim descobrir o valor final.
 								 
Aproveitei e adicionei mais uma funcionalidade com o CSS. Conforme passar o mouse nos tópicos de valores, ambos mudaram de cor. O valor de entrada fica verde-claro, em alusão a um ganho, enquanto o de saída fica vermelho, em alusão a uma retirada. E o valor total, fica em verde-escuro, simbolizando o valor bruto.
